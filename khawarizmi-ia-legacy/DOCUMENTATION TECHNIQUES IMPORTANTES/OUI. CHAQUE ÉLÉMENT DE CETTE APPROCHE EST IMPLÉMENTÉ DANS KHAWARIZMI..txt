## 4. "NE JAMAIS RESTER SUR JE SAIS PAS" → INDICES PROGRESSIFS

```
┌─────────────────────────────────────────────────────────────────┐
│                                                                 │
│  CE QUE DIT LE TEXTE :                                         │
│  "S'il bloque, il ne doit pas chercher la réponse             │
│   tout de suite."                                              │
│                                                                 │
│  CE QUE KHAWARIZMI FAIT :                                      │
│  Système d'indices à 3 niveaux AVANT la réponse.              │
│                                                                 │
│  ÉCHANGE 1 (élève bloque) :                                    │
│  IA : "Qu'est-ce que tu sais déjà sur ce sujet ?"            │
│  → Forcer la mémoire active (retrieval practice)              │
│                                                                 │
│  ÉCHANGE 2 (toujours bloqué) :                                 │
│  IA : "Pense à la formule du chapitre 3..."                  │
│  → Indice directionnel (pas la réponse)                       │
│                                                                 │
│  ÉCHANGE 3 (toujours bloqué) :                                 │
│  IA : "La première étape est : calcule d'abord U1.           │
│        Avec U0 = 3 et Un+1 = 2Un - 1, U1 = ?"               │
│  → Décomposition en sous-problèmes (scaffolding)             │
│                                                                 │
│  ÉCHANGE 4 (toujours bloqué) :                                 │
│  IA : "U1 = 2(3) - 1 = 5. Maintenant essaie U2."            │
│  → Donner la première étape, laisser finir                    │
│                                                                 │
│  ÉCHANGE 5 (toujours bloqué) :                                 │
│  IA : "Voici la solution complète avec explication..."        │
│  → Solution donnée SEULEMENT après 5 tentatives              │
│  → MAIS l'exercice reviendra dans 24h (SM-2 reset)           │
│                                                                 │
│  PERSONNE NE RESTE BLOQUÉ INDÉFINIMENT.                        │
│  MAIS PERSONNE N'OBTIENT LA RÉPONSE GRATUITEMENT.             │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

```javascript
// Implémentation du système d'indices progressifs

class ProgressiveHints {

    static async getHint(context, openai) {

        const level = context.exchangeCount;

        const hintStrategies = {
            1: `Pose une question ouverte : "Qu'est-ce que tu sais sur ce sujet ?"
                Ne donne aucun indice. Force la mémoire.`,

            2: `Donne un indice DIRECTIONNEL :
                "Regarde le chapitre sur ${context.chapitre}."
                "Quelle formule relie ${context.concepts[0]} et ${context.concepts[1]} ?"
                Ne donne PAS la formule.`,

            3: `Décompose le problème en sous-étapes :
                "D'abord, calcule [sous-problème 1]."
                "Ensuite, utilise ce résultat pour..."
                Donne la STRUCTURE, pas les calculs.`,

            4: `Donne la première étape complète :
                "La première étape donne : [résultat partiel]."
                "Maintenant, essaie la deuxième étape."`,

            5: `Donne la solution complète avec explication détaillée.
                MAIS ajoute : "Cet exercice reviendra demain pour
                que tu puisses le refaire seul."`,
        };

        const strategy = hintStrategies[Math.min(level, 5)];

        const prompt = `
${context.systemPrompt}

NIVEAU D'AIDE ACTUEL : ${level}/5
STRATÉGIE À APPLIQUER :
${strategy}

QUESTION : ${context.question}
RÉPONSE CORRECTE (pour toi, pas pour l'élève) : ${context.correctAnswer}
        `;

        return await openai.chat.completions.create({
            model: 'gpt-4o-mini',
            messages: [{ role: 'user', content: prompt }],
            temperature: 0.3,
            max_tokens: 300,
        });
    }
}
```
