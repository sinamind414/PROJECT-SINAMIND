الشفرة الوراثية

Genetic Code — لغة الحياة الجزيئية

💡 المفهوم العام

الشفرة الوراثية هي نظام مراسلة يربط بين اللغة النووية (4 قواعد) واللغة البروتينية (20 حمض أميني). الوحدة الأساسية هي الرامزة (Codon): تتابع 3 نيوكليوتيدات يُشفّر لحمض أميني واحد.

✨ خصائص الشفرة

  • عالمية: نفسها في كل الكائنات الحية تقريباً
  • تنكُّسية (تكرارية): عدة رامزات تشفر لنفس الحمض
  • غير متداخلة: قراءة متتالية بدون تداخل
  • محددة: كل رامزة تشفر لحمض واحد فقط
