# PHASE 2 — PROMPTS PRÊTS À L'EMPLOI

## 2.1 — Prompt QCM

```
Tu es un professeur expert en Sciences de la Vie et de la Terre 
pour le Bac algérien (terminale).

À partir du texte source ci-dessous, génère exactement 10 QCM.

CONTRAINTES OBLIGATOIRES :
- 4 propositions par question (A, B, C, D)
- Une seule bonne réponse par question
- Les 3 mauvaises réponses doivent être plausibles et proches du cours
- Pas de questions ambiguës
- Respecter le niveau Bac algérien
- Ne rien inventer en dehors du texte source
- Varier les formulations (Quel est... ? / Parmi... / Laquelle... ?)

FORMAT DE SORTIE (respecter exactement) :
---
Q[numéro] : [question]
A) [option A]
B) [option B]
C) [option C]
D) [option D]
✅ Réponse : [lettre]
💡 Justification : [explication courte de 1-2 lignes]
Niveau : [Facile / Moyen / Difficile]
Thème : [nom du sous-thème]
---

TEXTE SOURCE :
[coller l'extrait ici]
```

## 2.2 — Prompt Vrai/Faux

```
Tu es un professeur expert en Sciences de la Vie et de la Terre 
pour le Bac algérien (terminale).

À partir du texte source ci-dessous, génère exactement 10 
affirmations Vrai/Faux.

CONTRAINTES OBLIGATOIRES :
- Exactement 5 affirmations VRAIES et 5 affirmations FAUSSES
- Les affirmations fausses doivent être proches de la vérité 
  (erreur subtile, pas évidente)
- Chaque affirmation = une seule idée claire
- Éviter les affirmations trop longues
- Fournir la correction et la reformulation correcte 
  pour les affirmations fausses

FORMAT DE SORTIE :
---
Affirmation [numéro] :
📌 Énoncé : [l'affirmation]
✅ Réponse : VRAI / FAUX
🔄 Correction (si FAUX) : [la version correcte]
💡 Explication : [pourquoi c'est vrai ou faux]
Niveau : [Facile / Moyen / Difficile]
---

TEXTE SOURCE :
[coller l'extrait ici]
```

```

│
└── Soir (30 min) :
    ├── Vérification rapide
    └── Saisie dans le fichier Excel

Production journalière : 60 exercices
Production hebdomadaire : 300 exercices (5 jours)
```

# PHASE 5 — CONTRÔLE QUALITÉ

## Système de validation en 3 niveaux

```
NIVEAU 1 — Vérification automatique par IA
├── Envoyer 10 exercices à la fois
├── Utiliser le prompt de vérification
└── Marquer statut : ✅ / ⚠️ / ❌

NIVEAU 2 — Vérification croisée
├── Comparer avec le texte source
├── Vérifier la page et le chapitre
└── Vérifier qu'il n'y a pas de doublon

NIVEAU 3 — Relecture humaine
├── Analyser 10% des exercices (100 exercices)
├── Priorité : les analyses d'expériences
├── Priorité : les QCM difficiles
└── Corriger et mettre à jour le fichier
```

---

## Critères de qualité par type

### QCM — Liste de vérification
```
□ La question est claire et précise
□ Il y a exactement 4 propositions
□ Une seule bonne réponse
□ Les distracteurs sont plausibles
□ Pas d'indices dans la formulation
□ La justification est correcte
□ Le niveau est bien évalué
□ La source est indiquée
```

### Vrai/Faux — Liste de vérification
```
□ L'affirmation est courte et claire
□ Une seule idée par affirmation
□ Les FAUX sont subtilement faux
□ La correction est complète
□ Équilibre 50% VRAI / 50% FAUX
```

### Textes à trous — Liste de vérification
```
□ Les mots manquants sont des termes clés
□ Le contexte permet de deviner
□ La word bank est en désordre
□ La correction est complète
□ Pas trop de mots manquants (max 4)
```

### Analyses — Liste de vérification
```
□ Le contexte est réaliste
□ Le protocole est logique
□ Les résultats sont cohérents
□ Les questions progressent en difficulté
□ Le corrigé est complet et précis
□ Conforme au format Bac algérien
```

# PHASE 7 — ASTUCES POUR ALLER PLUS VITE

## Les 10 règles d'or

```
1️⃣  Toujours utiliser le même format de prompt
    → gain de temps : 40%

2️⃣  Générer par blocs de 500 mots maximum
    → meilleure qualité, moins d'erreurs

3️⃣  Demander toujours le format JSON ou tableau
    → copier-coller direct dans Excel

4️⃣  Générer 10% de plus que nécessaire
    → pour compenser les exercices à éliminer

5️⃣  Faire la vérification en batch (10 à la fois)
    → pas exercice par exercice

6️⃣  Utiliser des raccourcis clavier
    → préparer des textes de prompt en avance

7️⃣  Travailler chapitre par chapitre
    → ne pas mélanger les thèmes

8️⃣  Sauvegarder après chaque session
    → ne jamais perdre de travail

9️⃣  Garder un journal de production
    → savoir exactement où on en est

🔟  Améliorer les prompts chaque semaine
    → la qualité s'améliore avec le temps
```
